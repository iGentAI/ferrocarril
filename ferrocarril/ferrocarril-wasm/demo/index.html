Ferrocarril Wasm Demo

Pure-Rust Kokoro-82M TTS running inside your browser via wasm-bindgen. No ONNX Runtime, no server calls during inference.

1. Smoke test: load the wasm module

Initialises the wasm binary and constructs an empty WasmFerroModelBuilder to confirm the bindings are reachable. Does not fetch any weights.

(not run)

2. Full inference: fetch weights & synthesise

Loads the full 341 MB Kokoro weight pack from ./weights/, builds the model, and synthesises a short IPA utterance. The IPA must match Kokoro's vocabulary (see kokoro/tokenizer.py in the reference repo).

(not run)
