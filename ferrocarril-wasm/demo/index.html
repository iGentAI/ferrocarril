Ferrocarril

Pure-Rust port of Kokoro-82M running inside your browser via wasm-bindgen. No ONNX Runtime, no server round-trips during inference. Grapheme-to-phoneme, acoustic model, and vocoder all run in the same wasm module.

Heads up: the first visit downloads ~340 MB of model weights and takes tens of seconds. Subsequent visits are cached in your browser's IndexedDB and should start much faster. Inference itself takes roughly 4× wall time per second of audio on a recent desktop browser; expect a few seconds per short sentence.
